# 🚀 Installation Guide

## One-Line Installation & Demo

```bash
git clone https://github.com/Saytor20/Websites.git && cd Websites/Websites_nextjs && ./quick-start.sh
```

This will:
1. ✅ Clone the repository
2. ✅ Run setup automatically  
3. ✅ Generate a demo website
4. ✅ Launch preview at http://localhost:8080

## Step-by-Step Installation

### 1. Clone Repository
```bash
git clone https://github.com/Saytor20/Websites.git
cd Websites/Websites_nextjs
```

### 2. Run Setup
```bash
./setup.sh
```

### 3. Generate Your First Website
```bash
./websites
```
